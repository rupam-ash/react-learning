# 📂 React Accordion App

A simple and interactive Accordion component built using React.
This project helped me practice React state management, conditional rendering, event handling, and dynamic UI updates.

---

## 🚀 Features

* 📌 Expand and collapse accordion items
* 🔄 Toggle functionality
* 🎯 Only one accordion stays open at a time
* ✨ Dynamic icons (`+` / `-`)
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* React
* JavaScript (ES6)
* CSS
* Vite

---

## 📂 Project Structure

```bash
src/
 ├── components/
 │    └── AccordianItem.jsx
 │
 ├── Accordian.jsx
 ├── App.jsx
 ├── main.jsx
 └── index.css
```

---

## ⚙️ How It Works

* Accordion data is stored in an array of objects
* Clicking an item updates the currently open accordion state
* Conditional rendering displays either the question or answer
* Dynamic classes are used for styling open items
